# atm
Multiple Persons Bank Account Transaction System
## 1. ATM - Automated Teller Machine coded in python. 

This machine works for multiple bank account handling. Where Each persons data will be saved in seperate
txt files. 

### Features:
1. Sign Up 
2. Log in
3. View Account
4. Change pin
5. Withdraw 
6. Deposit 
7. Pay bills 
8. View history
